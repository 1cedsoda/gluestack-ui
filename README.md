## react-native-aria

## Run

1. Install lerna and tsc globally

2. Generate lib.

```
   cd [project-root]
   lerna tsc
```

3. Install dependencies

```
   yarn
```

4. Run example

```
   cd [project-root]/example
   yarn

   // Web
   yarn web

   // iOS
   yarn ios

   // Android
   yarn android
```
