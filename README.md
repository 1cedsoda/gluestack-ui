## react-native-aria

## Installation

1. Install project dependencies

   ```
   cd [project-root]
   yarn
   ```

2. Install example project dependencies
   ```
   cd [project-root]/example/
   yarn
   ```

## Run

Web

```
 cd [project-root]
 yarn example web
```

iOS

```
 cd [project-root]
 yarn example ios
```

Android

```
 cd [project-root]
 yarn example android
```
