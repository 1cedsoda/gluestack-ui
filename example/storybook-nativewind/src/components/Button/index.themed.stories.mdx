## gluestack-style

Step 1: Install the following dependencies:

```bash

npm install @gluestack-style/react
```

# Usage

```jsx
import { Button } from '@/components/ui/button';
```

```jsx
<Button variant="outline">
  <ButtonText>Button</ButtonText>
</Button>
```
