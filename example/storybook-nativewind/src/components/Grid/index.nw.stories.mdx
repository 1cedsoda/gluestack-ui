## API Reference

To use this component in your project, include the following import statement in your file.

```jsx
import { Grid, GridItem } from '@/components/ui/grid';
```

```jsx
export default () => (
<Grid >
<GridItem />
</Grid>
);
```

### Spacing

To control the space between the grid items, use the `spacing` prop.
The `spacing` prop takes an integer value between 0 and 12. The default value is 0.

<Wrapper>
  <CodePreview
    showComponentRenderer={true}

      code: `

      <GridItem
        colSpan={3}

        <Text>A</Text>
      </GridItem>
