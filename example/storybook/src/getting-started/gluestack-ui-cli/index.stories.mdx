### Commands provided by gluestack-ui

`gluestack-ui` provides a CLI to help you get started with the library. It includes initial installation setup and adding, updating or removing components in your project. Below is the list of commands provided by `gluestack-ui`.

```bash

# init gluestack-ui
npx gluestack-ui@latest init

# init and add components
npx gluestack-ui@latest

# add component
npx gluestack-ui@latest add <component-name>

# update component
npx gluestack-ui@latest update <component-name>

# remove component
npx gluestack-ui@latest remove <component-name>

# help
npx gluestack-ui@latest help
```
