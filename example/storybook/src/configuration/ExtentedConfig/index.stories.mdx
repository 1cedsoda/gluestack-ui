# Configure your Extended Config

To configure extended config in @gluestack-style/react, create a gluestack-style.config.js file to define your extended theme along with other configuration settings.

```jsx
import { createConfig } from "@gluestack-style/react"
export const config = createConfig({
  aliases: {},
  tokens: {},
   components: {
    Button: {  // write same component name which is passed in styled
      theme: {
        'bg': '$primary500',
        variants:{
          variant:{
            secondary:{
              bg:'$secondary500',
            }
          }
        }
      },
      componentConfig: {
        descendantStyle: ['_text'],
      },
    },
  },
} as const);

```

### To use extended config

```jsx
const MyButton = styled(Pressable, {
  bg: '$red500',
  p: '$3',
},
{
  componentName:"Button" // pass same component name which is passed in config
});

const export Button = () => <MyButton  variant="secondary"/>;
``
```
