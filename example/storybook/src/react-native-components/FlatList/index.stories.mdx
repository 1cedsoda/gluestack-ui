### Import

To use the component in your project, run the command below into your import statement. This will import the component from your components folder.

```bash
import { FlatList } from '@gluestack-ui/themed';
```

### Customizing the FlatList

For customizing the FlatList, you have to map styled view to FlatList. You can refer [style.gluestack.io](https://style.gluestack.io/) for more information on how to use styled components.

#### Usage

Default styling of the component can be found in the `components/FlatList` file. For reference, you can view the [source code](https://github.com/gluestack/gluestack-ui/tree/main/packages/themed/src/components/FlatList/styled-components) of the styled `FlatList` component.

```jsx
// import the styles
import { Root } from '../components/core/FlatList/styled-components';

// mapping the FlatList to root component
const FlatList = Root;

// using the FlatList component
export default () => <FlatList />;
```
