<br />

## Features

- support of props on any svg icon

## Installation

```jsx
yarn add @universa11y/icon
```

## Usage

```jsx
import { createIcon } from '@universa11y/icon';
const Icon = createIcon({
  Root,
  viewBox,
  d,
});
```

These are the styled components that we need to pass in the creator API:

### Root

```jsx
const Root = styled(
  Svg,
  {
    ...
  },
);
```

## Anatomy

```jsx
export default () => <AddIcon />;
```

## API Reference

### Icon

It inherits all the properties of React Native SVG's [svg](https://github.com/software-mansion/react-native-svg/tree/main/Example) component.
