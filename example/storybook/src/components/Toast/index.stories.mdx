# Toast

Toast is a component that allows you to add an string value in a paragraph to an input.

## Usage

```jsx
import { Toast } from '@universa11y/textarea';
```

```jsx
<Toast id="toast">
  <Toast.Title>Toast Title</Toast.Title>
  <Toast.Description>Toast Body</Toast.Description>
</Toast>
```
