# Alert-dialog

Alert dialogs interrupt users with urgent information, details, or actions.
In React Native, the Alert-dialog component is typically generated using "styled views" and "pressables".

The styles of Gluestack Ui Alert-dialog are injected from [@dank-style/react]('https://www.npmjs.com/package/@dank-style/react') and accessibility and other functionality is provided by [@gluestack-ui/alert-dialog]('https://www.npmjs.com/package/@gluestack-ui/alert-dialog').

## Import

```jsx
import { AlertDialog } from './components/alert-dialog';
```

## Usage

              'Return button. Please make a note of the Return ID that we

            </Text>
          </AlertDialog.Body>
          <AlertDialog.Footer>
            <Button style="solid" onPress={handleClose}>
              <Button.Text>Cancel</Button.Text>
            </Button>
          </AlertDialog.Footer>
        </AlertDialog.Content>
      </AlertDialog>
    </Wrapper>
    )
    }`,
      scope: {
        Wrapper,
        AlertDialog,
        Button,
        Text,
        CloseIcon,
      },
      argsType: {},
    }}
  />
</AppProvider>
