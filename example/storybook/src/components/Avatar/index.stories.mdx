# Avatar

Use the Avatar Component as a graphical representation of a person through a profile picture or image, an icon, or a string with initials.

In React Native, the Avatar component is typically generated using "styled views" and "pressables".

The styles of Gluestack Ui Avatar are injected from [@dank-style/react]('https://www.npmjs.com/package/@dank-style/react') and accessibility and other functionality is provided by [@universa11y/avatar]('https://www.npmjs.com/package/@universa11y/avatar').

## Import

```jsx
import { Avatar } from './components/avatar';
```
