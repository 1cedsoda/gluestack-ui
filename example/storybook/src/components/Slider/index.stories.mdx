# Slider

A slider component is a graphical user interface element that allows the user to select a value from a range by sliding a knob along a track.

<AppProvider>
  <CodePreview
    showComponentRenderer={true}
    showArgsController={false}
    metaData={{
      code: ` 
       const [sliderValue, setSliderValue] = React.useState(40);
  const handleChange = (value: any) => {
    setSliderValue(value);
  };
      function App () {
        return (
       <Slider
        w="50%"
        value={sliderValue}
        onChange={(value: any) => {
          handleChange(value);
        }}
      >
        <Slider.Track>
          <Slider.FilledTrack />
        </Slider.Track>
        <Slider.Thumb />
      </Slider>
    )
    }`,
      scope: { Slider: AccessibleSlider, Text },
      argsType: {},
    }}
  />
</AppProvider>

<br />

## Features

- Can be controlled or uncontrolled.
- Full keyboard navigation.

## Installation

```jsx
yarn add @universa11y/slider
```

## Usage

```jsx
import { createSlider } from '@universa11y/slider';
const Slider = createSlider({
  Root,
  ThumbInteraction,
  Thumb,
  Track,
  FilledTrack,
});
```

## API Reference

### Slider

It inherits all the properties of React Native's [view](https://reactnative.dev/docs/view#props) component.

| Prop               | Type                       | Default    | Description |
| ------------------ | -------------------------- | ---------- | ----------- |
| value              | number                     | 0          | -           |
| defaultValue       | number                     | 0          | -           |
| onChange           | function                   |            | -           |
| accessibilityLabel | string                     | Slider     | -           |
| isReversed         | bool                       | false      | -           |
| orientation        | 'horizontal' or 'vertical' | horizontal | -           |
| isDisabled         | bool                       | false      | -           |
| onChangeEnd        | function                   |            | -           |
| minValue           | number                     | 0          | -           |
| step               | number                     | 1          | -           |
| isReadOnly         | bool                       | false      | -           |
| thumbSize          | number                     | false      | -           |

### Slider.Thumb

Contains all thumb related layout style props and actions.
It inherits all the properties of React Native's [Pressable](https://reactnative.dev/docs/pressable#props) component.

| Prop        | Type                       | Default      |
| ----------- | -------------------------- | ------------ |
| orientation | 'horizontal' or 'vertical' | 'horizontal' |
| isDisabled  | bool                       | false        |
| isReadOnly  | bool                       | false        |
| onFocus     | function                   |              |
| onBlur      | function                   |              |

### Slider.Track

Contains all track related layout style props and actions.
It inherits all the properties of React Native's [Pressable](https://reactnative.dev/docs/pressable#props) component.

| Prop       | Type | Default |
| ---------- | ---- | ------- |
| isReadOnly | bool | false   |

### Slider.FilledTrack

Contains all filled track view related layout style props and actions.
It inherits all the properties of React Native's [View](https://reactnative.dev/docs/view#props) component.

| Prop       | Type | Default |
| ---------- | ---- | ------- |
| isReadOnly | bool | false   |
