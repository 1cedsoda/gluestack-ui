# Slider

A slider component is a graphical user interface element that allows the user to select a value from a range by sliding a knob along a track.

<AppProvider>
  <CodePreview
    showComponentRenderer={true}
    showArgsController={false}
    metaData={{
      code: ` 
      function App () {
          const [sliderValue, setSliderValue] = useState(40);
          const handleChange = (value) => {
            setSliderValue(value);
          };
        return (
          <Wrapper>
            <Slider
        w="300"
        value={sliderValue}
        onChange={(value) => {
          handleChange(value);
        }}
      >
        <Slider.Track>
          <Slider.FilledTrack />
        </Slider.Track>
        <Slider.Thumb />
      </Slider></Wrapper>
    )
    }`,
      scope: { Slider: AccessibleSlider, Text, Wrapper, useState },
      argsType: {},
    }}
  />
</AppProvider>

<br />

## Features

- Can be controlled or uncontrolled.
- Full keyboard navigation.

## Installation

```jsx
yarn add @universa11y/slider
```

## Usage

```jsx
import { createSlider } from '@universa11y/slider';
const Slider = createSlider({
  Root,
  ThumbInteraction,
  Thumb,
  Track,
  FilledTrack,
});
```
