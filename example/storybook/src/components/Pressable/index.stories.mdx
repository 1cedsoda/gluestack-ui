# Pressable

A Pressable component is used to create clickable elements that perform an action when pressed.

<br />

## Features

- Keyboard support for actions.
- Support for hover, focus and active states.

## Installation

```jsx
yarn add @universa11y/pressable
```

## Usage

```jsx
import { createPressable } from '@universa11y/pressable';
const Pressable = createPressable({
  Root,
});
```

These are the styled components that we need to pass in the creator API:

### Root

```jsx
const Root = styled(
  Pressable,
  {
    ...
  },
);
```

## Anatomy

```jsx
export default () => (
  <Pressable>
    <Text></Text>
  </Pressable>
);
```

## API Reference

### Pressable

Contains all Pressable related layout style props and actions.
It inherits all the properties of React Native's [Pressable](https://reactnative.dev/docs/pressable#props) component.

## Accessibility

### Keyboard

- Tab: Moves focus to the next focusable element.
- `Shift + Tab`: Moves focus to the previous focusable element.
- Enter: Triggers the Pressable's action.

### Screen Reader

- VoiceOver: When the Pressable is focused, the screen reader will announce the Pressable's label and its current state.
