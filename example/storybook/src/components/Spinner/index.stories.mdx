# Spinner

A Spinner bar is a graphical control element used to visualize the Spinnerion of an extended computer operation, such as a download, file transfer, or installation.

## Features

- Keyboard support for actions.
- Support for hover, focus and active states.

## Installation

```jsx
yarn add @universa11y/Spinner
```

## Usage

```jsx
import { createSpinner } from '@universa11y/Spinner';
const Spinner = createSpinner({
  Root,
});
```

These are the styled components that we need to pass in the creator API:

### Root

```jsx
const Root = styled(
  Spinner,
  {
    ...
  },
);
```

## Anatomy

```jsx
export default () => (
  <Spinner>
    <Text></Text>
  </Spinner>
);
```

## API Reference

### Spinner

Contains all Spinner related layout style props and actions.
It inherits all the properties of React Native's [Spinner](https://reactnative.dev/docs/Spinner#props) component.

## Accessibility

### Keyboard

- Adheres to the Spinnerbar [role requirements]("https://www.w3.org/TR/wai-aria-1.2/#Spinnerbar").

- On web, aria-value now, aria-value min and aria-value max is used to ensure the Spinner percent is visible to screen readers.

- On mobile, [accessibilityValue]('https://reactnative.dev/docs/accessibility#accessibilityvalue') is used to ensure it is announced by Talkback and VoiceOver.
