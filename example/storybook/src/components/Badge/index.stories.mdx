# Badge

Alert dialogs interrupt users with urgent information, details, or actions.
In React Native, the Badge component is typically generated using "styled views" and "pressables".

The styles of Gluestack Ui Badge are injected from [@dank-style/react]('https://www.npmjs.com/package/@dank-style/react') and accessibility and other functionality is provided by [@universa11y/badge]('https://www.npmjs.com/package/@universa11y/badge').

## Import

```jsx
import { Badge } from './components/badge';
```

## Usage

<AppProvider>
