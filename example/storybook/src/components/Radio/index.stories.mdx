# Radio

A Radio component is used to provide a way for users to select one option from a set of predefined choices. In React Native, the Radio component can be generated using a third-party accessibility-focused library that combines "styled views" and "pressable" to define the layout, visual design, and behavior of the component.

## Features

- Keyboard support for actions.
- Support for hover, focus and active states.

## Installation

```jsx
yarn add @universa11y/radio
```

## Usage

```jsx
import { createRadio } from '@universa11y/radio';
const Radio = createRadio({
  Root,
  Indicator,
  Icon,
  Label,
  Group,
});
```

These are the styled components that we need to pass in the creator API:

## Acceessibility

### Keyboard

- Tab: Moves focus to the next focusable element.
- `Shift + Tab`: Moves focus to the previous focusable element.
- Space: To check or uncheck focused radio.
- Keydown: Moves selected radio button inside a radio group.

### Screen Reader

- VoiceOver: When the radio is focused, the screen reader will announce it's a radio button and it's current state (unselected and selected) and it's label.
