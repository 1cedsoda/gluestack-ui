# Select

A Select component is used to create clickable elements that perform an action when pressed.

## Features

- Support for native and web platforms separately.
- Support for passing custom trigger.

## Installation

```jsx
yarn add @universa11y/select
```

## Usage

Select is a component that need Actionsheet to work on native. So we are passing Actionsheet as a second argument or dependency to the creator API. We can also pass different styled components for actionsheet.

```jsx
import { createSelect } from '@universa11y/select';
import { createActionsheet } from '@universa11y/actionsheet';

const Actionsheet = createActionsheet({
  Root: ActionsheetRoot,
  Backdrop,
  Item: ActionsheetItem,
  ItemText,
  DragIndicator,
  IndicatorWrapper,
  Content,
});

const Select = createSelect(
  {
    Root,
    Icon,
    Item,
    ItemList,
  },
  { Actionsheet }
);
```

These are the styled components that we need to pass in the creator API:

For actionsheet specific styled components, look into the [actionsheet docs](https://universa11y.dev/docs/actionsheet).

```jsx
const Root = styled(
  View,
  {
    ...
  },
);
```
