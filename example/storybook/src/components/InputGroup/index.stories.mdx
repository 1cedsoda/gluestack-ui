# InputGroup

InputGroup is a component that allows you to add an icon or button to an input.

## Usage

import { InputGroup } from '@universa11y/input';

```jsx
<InputGroup id="input-group" placeholder="Placeholder" />
```

## Props

| Name        | Type   | Default | Description                  |
| ----------- | ------ | ------- | ---------------------------- |
| id          | string |         | The id of the input          |
| placeholder | string |         | The placeholder of the input |
