# Installation

This command copies the `Link` component to your project.

```jsx
npx install @gluestack-ui add Link
```

## Usage

```jsx
// import the component from your component folder
import { Link } from 'components';

// Using the link component
export default () => (
  <Link href="https://gluestack.io/">
    <Text>Link</Text>
  </Link>
);
```

# Advanced

## Customizing the Link

We have a function called `createLink` which can be used to create a custom link component. This function takes in a configuration object which contains the styled components that you want to use for the link. You can refer [dank.style](https://dank.style/) for more information on how to use styled components.

### Usage

```jsx
// import the styles
import { Root } from '../ui-components/Link/styled-components';

// import the createLink function
import { createLink } from '@gluestack-ui/link';

// Understanding the API
const Link = createLink({
  Root,
});

// Using the link component
export default () => (
  <Link href="">
    <Text />
  </Link>
);
```

Default styling of all these components can be found in the `ui-components/Link` file. For reference, you can view the [source code](https://github.com/gluestack/gluestack-ui/blob/development/example/storybook/src/ui-components/Link/index.tsx) of the styled `Link` components.
