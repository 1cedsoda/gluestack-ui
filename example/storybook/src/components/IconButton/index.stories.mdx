# IconButton

A IconButton component is used to create clickable elements that perform an action when pressed with icons.

<AppProvider>
  <CodePreview

        <IconButton.Spinner />

          </Wrapper>
    )
    }`,
      scope: { IconButton: AccessibleIconButton, Wrapper },
      argsType: {},
    }}
  />
</AppProvider>

<br />

## Features

- Keyboard support for actions.
- Support for hover, focus and active states.

## Installation

```jsx
yarn add @universa11y/button
```

## Usage

```jsx
import { createButton } from '@universa11y/button';
const IconButton = createIconButton({
  Root,
  Text,
  Spinner,
});
```

These are the styled components that we need to pass in the creator API:

### Root

```jsx
const Root = styled(
  Pressable,
  {
    ...
  },
);
```

### Text

```jsx
import { styled } from '@dank-style/react';
const Text = styled(
  Text,
  {
    ...
  },
);
```

### Spinner

```jsx
import { styled } from '@dank-style/react';
const Spinner = styled(
  ActivityIndicator,
  {
    ...
  },
);
```

## Anatomy

```jsx
export default () => (
  <IconButton>
    <IconButton.Text />
    <IconButton.Spinner />
  </IconButton>
);
```

## API Reference

### IconButton

Contains all button related layout style props and actions.
It inherits all the properties of React Native's [Pressable](https://reactnative.dev/docs/pressable#props) component.

### IconButton.Text

Contains all text related layout style props and actions.
It inherits all the properties of React Native's [Text](https://reactnative.dev/docs/text#props) component.

### IconButton.Spinner

Contains all spinner related layout style props and actions.
It inherits all the properties of React Native's [ActivityIndicator](https://reactnative.dev/docs/activityindicator#props) component.

## Accessibility

### Keyboard

- Tab: Moves focus to the next focusable element.
- `Shift + Tab`: Moves focus to the previous focusable element.
- Enter: Triggers the button's action.

### Screen Reader

- VoiceOver: When the button is focused, the screen reader will announce the button's label and its current state.
