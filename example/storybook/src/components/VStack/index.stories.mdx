# VStack

A `VStack` component is used to create a vertical stack layout for child components.

<AppProvider>
  <CodePreview
    showComponentRenderer={true}
    showArgsController={false}
    metaData={{
      code: ` 
      function App () {
        return (<Wrapper><VStack
        space="md"
        sx={{ justifyContent: 'center', alignItems: 'center' }}
      >
        <View
          style={{ height: 20, width: 20, backgroundColor: 'red' }}
        />
        <View
          style={{ height: 20, width: 20, backgroundColor: 'red' }}
        />
        <View
          style={{ height: 20, width: 20, backgroundColor: 'red' }}
        />
        <View
          style={{ height: 20, width: 20, backgroundColor: 'red' }}
        />
      </VStack></Wrapper>)
    }`,
      scope: { VStack: AccessibleVStack, View, Wrapper },
      argsType: {},
    }}
  />
</AppProvider>

<br />

## Features

- Helpful in layouting components in a vertical manner and also helps in giving equal space between the components.

## Installation

```jsx
yarn add @universa11y/VStack
```

## Usage

```jsx
import { createVStack } from '@universa11y/VStack';
const VStack = createVStack({
  Root,
  Spacer,
});
```

These are the styled components that we need to pass in the creator API:

### Root

```jsx
const Root = styled(
  View,
  {
    ...
  },
);
```

### Root

```jsx
const Spacer = styled(
  View,
  {
    ...
  },
);
```

## Anatomy

```jsx
export default () => (
  <VStack>
    <View />
    <View />
    <View />
  </VStack>
);
```
