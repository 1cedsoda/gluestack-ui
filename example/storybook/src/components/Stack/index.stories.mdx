# Stack

A `Stack` component is used to create a horizontal or vertical stack layout for child components.

<AppProvider>
  <CodePreview

        sx={{ justifyContent: 'center', alignItems: 'center' }}

          style={{ height: 20, width: 20, backgroundColor: 'red' }}
        />
        <View
          style={{ height: 20, width: 20, backgroundColor: 'red' }}
        />
        <View
          style={{ height: 20, width: 20, backgroundColor: 'red' }}
        />
        <View
          style={{ height: 20, width: 20, backgroundColor: 'red' }}

      </Stack></Wrapper>)
    }`,
      scope: { Stack: AccessibleStack, View, Wrapper },
      argsType: {},
    }}
  />
</AppProvider>

<br />

## Features

- Helpful in layouting components in a horizontal or vertical manner and also helps in giving equal space between the components.

## Installation

```jsx
yarn add @universa11y/Stack
```

## Usage

```jsx
import { createStack } from '@universa11y/Stack';
const Stack = createStack({
  Root,
  VSpacer,
  HSpacer,
});
```

These are the styled components that we need to pass in the creator API:

### Root

```jsx
const Root = styled(
  View,
  {
    ...
  },
);
```

### VSpacer

```jsx
const Spacer = styled(
  View,
  {
    ...
  },
);
```

### HSpacer

```jsx
const Spacer = styled(
  View,
  {
    ...
  },
);
```

## Anatomy

```jsx
export default () => (
  <Stack>
    <View />
    <View />
    <View />
  </Stack>
);
```

## API Reference

### Stack

Contains all View related layout style props and actions.
It inherits all the properties of React Native's [View](https://reactnative.dev/docs/view#props) component.
