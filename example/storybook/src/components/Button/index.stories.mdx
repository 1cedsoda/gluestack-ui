# Button

A Button component is used to create clickable elements that perform an action when pressed.

The styles of Gluestack Ui button is injected from [@dank-style/react]('https://www.npmjs.com/package/@dank-style/react') and accessibility and other functionality is provided by [@gluestack-ui/button]('https://www.npmjs.com/package/@universa11y/button').

## Features

- Keyboard support for actions.
- Support for hover, focus and active states.
- Option to add your styles or use the default styles.

## Installation

```jsx
yarn add @gluestack-ui/button
```

## Import

```jsx
import { createButton } from '@gluestack-ui/button';
```

## Understanding the API

```jsx
const Button = createButton({
  Root,
  Text,
  Group,
  GroupHSpacer,
  GroupVSpacer,
  Spinner,
});
```

It gives you a button structure with all the functionality and accessibility.
Here, Root is the root component of the button, Text is the text component of the button, Group is the group component of the button, GroupHSpacer is the horizontal spacer component of the button, GroupVSpacer is the vertical spacer component of the button and Spinner is the spinner component of the button.
You can pass your own components to the API to create your own button.

## Anatomy

```jsx
export default () => (
  <Button.Group>
    <Button>
      <Button.Text />
      <Button.Spinner />
    </Button>
  </Button.Group>
);
```

## Usage (Unstyled)

<AppProvider>
  <CodePreview
    showComponentRenderer={true}
    showArgsController={false}
    metaData={{
      code: `
      function App() {
          const Root = ({ children, states, ...props }) => {
          return (
            <Pressable
              style={{
                backgroundColor: states.hover ? 'blue' : 'pink',
                padding: 8,
                borderRadius: 4,
              }}
              {...props}
            >
              {children}
            </Pressable>
          );
        };
        const ButtonText = ({ children, ...props }) => {
          return (
            <RNText style={{ color: 'white' }} {...props}>
              {children}
            </RNText>
          );
        };
        const ButtonGroup = ({ children, ...props }) => {
          return <View {...props}>{children}</View>;
        };
        const ButtonGroupSpacer = ({ children, ...props }) => {
          return <View {...props}>{children}</View>;
        };
        const ButtonSpinner = ({ children, ...props }) => {
          return <ActivityIndicator {...props}>{children}</ActivityIndicator>;
        };
        const ExampleButton = createButton({
          Root: Root,
          Text: ButtonText,
          Group: ButtonGroup,
          GroupHSpacer: ButtonGroupSpacer,
          GroupVSpacer: ButtonGroupSpacer,
          Spinner: ButtonSpinner,
        });
        return (
          <Wrapper>
          <ExampleButton>
          <ExampleButton.Text>Button</ExampleButton.Text>
          </ExampleButton>
          </Wrapper>
          );
        }
`,
      scope: {
        Wrapper: Wrapper,
        ActivityIndicator,
        View,
        Pressable,
        RNText,
        createButton,
      },
      argsType: {},
    }}
  />
</AppProvider>

## Accessibility

### Keyboard

- Tab: Moves focus to the next focusable element.
- `Shift + Tab`: Moves focus to the previous focusable element.
- Enter: Triggers the button's action.

### Screen Reader

- VoiceOver: When the button is focused, the screen reader will announce the button's label and its current state.
