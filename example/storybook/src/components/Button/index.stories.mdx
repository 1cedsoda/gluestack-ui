# Button

A Button component is used to create clickable elements that perform an action when pressed.

      scope: { Button: AccessibleButton, Wrapper },
      argsType: {},
    }}
  />
</AppProvider>

<br />

## Features

- Keyboard support for actions.
- Support for hover, focus and active states.

## Installation

```jsx
yarn add @universa11y/button
```

## Usage

```jsx
import { createButton } from '@universa11y/button';
const Button = createButton({
  Root,
  Text,
  Group,
  GroupSpacer,
  Spinner,
});
```

These are the styled components that we need to pass in the creator API:

### Root

```jsx
const Root = styled(
  Pressable,
  {
    ...
  },
);
```

### Text

```jsx
import { styled } from '@dank-style/react';

  },
```

## Accessibility

### Keyboard

- Tab: Moves focus to the next focusable element.
- `Shift + Tab`: Moves focus to the previous focusable element.
- Enter: Triggers the button's action.

### Screen Reader

- VoiceOver: When the button is focused, the screen reader will announce the button's label and its current state.
