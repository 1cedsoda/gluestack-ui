# Input

The Input component allows a user to provide input in a text field.

<AppProvider>
  <CodePreview
    showComponentRenderer={true}
    showArgsController={false}
    metaData={{
      code: ` 
      function App () {
  const [value, setValue] = useState('Some Random Text');
        return (
          <Wrapper>
      <InputField>
        <InputField.Input
          onChangeText={(text) => {
            setValue(text);
          }}
          value={value}
          placeholder="Enter Text here"
        />
      </InputField>
    </Wrapper>
    )
    }`,
      scope: { InputField: AccessibleInput, Wrapper, useState },
      argsType: {},
    }}
  />
</AppProvider>

<br />

## Features

- Keyboard support for actions.
- Support for hover, focus and active states.

## Installation

```jsx
yarn add @universa11y/input
```

## Usage

```jsx
import { createButton } from '@universa11y/button';
const Input = createInput({
  Root,
  Icon,
  Input,
});
```

These are the styled components that we need to pass in the creator API:

### Root

```jsx
const Root = styled(
  View,
  {
    ...
  },
);
```

### Icon

```jsx
import { styled } from '@dank-style/react';
const Icon = styled(
  Svg,
  {
    ...
  },
);
```

### Input

```jsx
import { styled } from '@dank-style/react';
const Group = styled(
  TextInput,
  {
    ...
  },
);
```

## Anatomy

```jsx
export default () => (
  <InputField>
    <InputField.Icon />
    <InputField.Input />
  </InputField>
);
```

## API Reference

### InputField

Contains all wrapper related layout style props and actions.
It inherits all the properties of React Native's [View](https://reactnative.dev/docs/view#props) component.

### InputField.Input

Contains all text related layout style props and actions.
It inherits all the properties of React Native's [TextInput](https://reactnative.dev/docs/TextInput#props) component.

## Accessibility

### Keyboard

- setting the accessibilityLabel and accessibilityHint

### Screen Reader

- VoiceOver: accessible and accessibilityLabel props to describe the input's purpose
- `accessibilityTraits` and `accessibilityHint` for the various states of the input, such as "double tap to edit"

## Dependencies

- `@universa11y/utils`
- `@react-native-aria/focus`
- `react-native-svg`
- `@universa11y/react-native-aria`
