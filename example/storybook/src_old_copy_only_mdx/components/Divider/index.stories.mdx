# Divider

A Divider component is used to create a visual separation between different sections or elements in a layout.

<AppProvider>
  <CodePreview
    showComponentRenderer={true}
    showArgsController={false}
    metaData={{
      code: ` 
      function App () {
        return (
          <Wrapper>
            <View style={{width:100}}>
              <Divider/>
            </View>
          </Wrapper>
          )
    }`,
      scope: { Divider: AccessibleDivider, View, Wrapper },
      argsType: {},
    }}
  />
</AppProvider>

<br />

## Features

- Can create variants of the Divider component by using the `variant` or `orientation` prop.

## Installation

```jsx
yarn add @universa11y/divider
```

## Usage

```jsx
import { createDivider } from '@universa11y/divider';
const Divider = createDivider({
  Root,
});
```

These are the styled components that we need to pass in the creator API:

### Root

```jsx
const Root = styled(
  View,
  {
    ...
  },
);
```

## Anatomy

```jsx
export default () => <Divider />;
```

## API Reference

### Divider

Contains all View related layout style props and actions.
It inherits all the properties of React Native's [View](https://reactnative.dev/docs/view#props) component.
